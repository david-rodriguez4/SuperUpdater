Acerca de

SuperUpdater es una aplicación de escritorio para Windows que permite gestionar y actualizar el software instalado en el sistema usando winget, el gestor de paquetes oficial de Microsoft. Desarrollada con Tauri v2 y Rust para máxima eficiencia y bajo consumo de recursos.
